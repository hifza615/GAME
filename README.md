Animated Hero Game 🎮

An interactive and visually engaging superhero-themed game built using HTML, CSS, and JavaScript. This project was created to practice front-end web development and basic game mechanics while bringing animated characters and a custom game environment to life.

The game features character animations, movement controls, and a dynamic layout designed to provide a fun and enjoyable experience. With a simple and beginner-friendly code structure, the project demonstrates how HTML, CSS, and JavaScript can be combined to create interactive web games without relying on external frameworks.

✨ Features
• Smooth character animations
• Movement controls and interactive gameplay
• Custom backgrounds and game elements
• Responsive and organized design
• Built entirely with HTML, CSS, and JavaScript
• Beginner-friendly and easy to understand
• Great project for learning front-end development and JavaScript fundamentals

🚀 Technologies Used
- HTML5
- CSS3
- JavaScript
